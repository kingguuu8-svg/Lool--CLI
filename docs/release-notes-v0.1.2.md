# Lool--CLI v0.1.2

## What it is

Lool--CLI is a self-hosted launcher for local coding agent CLIs on Android, desktop, browser, and VPS.

## Who it is for

- People who want to keep their coding agent CLI on their own machine
- People who want a quick Android entry point into a full-screen terminal UI
- People who want a Windows launcher that hides the batch-file setup

## Quick start

1. Download the Windows launcher from the release assets
2. Run it once and pick the project root if needed
3. Start local, LAN, VPS, or desktop mode with one click
4. Install the Android APK if you want a phone entry point

## Current limitations

- Android is still a shell around the browser terminal page
- Public access still depends on the VPS or tunnel setup
- The launcher is Windows-only for now
