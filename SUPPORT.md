# Support

PocketCLI is currently maintained as a self-hosted personal project that is being prepared for broader open-source use.

## Best way to get help

- open a GitHub issue for bugs, docs gaps, and feature requests
- include your mode: local, LAN, temporary public tunnel, or VPS
- include your platform: Windows, Android, or both
- include the exact startup script you used

## Good support details

- what you expected
- what actually happened
- relevant logs with secrets removed
- screenshots for UI problems
- whether the browser UI works when Android does not

## Before opening an issue

- read the [README](README.md)
- check the [Chinese guide](docs/zh-CN.md) if that is easier for you
- make sure you did not commit or paste secrets into your report
